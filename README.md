# Quote of the Day 💬

A simple quote app built with vanilla HTML, CSS, and JavaScript.

## Features

- Click "New Quote" to get a random motivational quote
- Shows the author of each quote
- Tracks how many quotes you've seen
- Copy any quote to clipboard with one click

## What I Practised

- DOM manipulation
- Event listeners
- Arrays and objects
- Math.random()
- classList methods
- setTimeout

## How to Run

Just open `index.html` in your browser — no installation needed!
